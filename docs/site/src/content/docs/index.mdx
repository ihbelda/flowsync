FlowSync es el starter kit full-stack (**AdonisJS 7 + React 19**) del máster
**AI4Devs**. Usa el menú lateral para navegar por la documentación:

- **Decisiones (ADRs)** — registro de las decisiones de arquitectura del
  proyecto y su justificación.
- **Guides** / **Reference** — plantillas de ejemplo del starter kit de
  Starlight, pendientes de completar con contenido propio.

Repositorio: [github.com/ihbelda/flowsync](https://github.com/ihbelda/flowsync)
